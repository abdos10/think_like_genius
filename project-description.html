ThinkTrainer

An AI-powered web application to teach and improve thinking skills

Project Overview

ThinkTrainer is a comprehensive web application designed to help users develop and enhance various thinking skills through interactive exercises, guided practice, and AI-powered feedback. The platform provides personalized training paths, tracks progress over time, and offers insights to help users become more effective thinkers.

Project Goal: Create a platform that makes developing critical thinking, creative thinking, analytical thinking, and strategic thinking accessible to everyone through personalized, AI-guided practice.

Architecture Overview

Application Structure

  • Client: React frontend with TailwindCSS for styling and shadcn components
  • Server: Express API with endpoints for data management and OpenAI integration
  • Shared: Common types and schemas shared between client and server
  • Storage: In-memory data store with option to migrate to PostgreSQL

Data Flow

The application follows a standard client-server architecture with React Query for data fetching and state management. User actions trigger API requests to the backend, which then interacts with the storage layer or external services (OpenAI API).

Key Components

Dashboard

The central hub that displays personalized insights through visualizations like the skills radar chart, weekly activity trends, and recommended exercises. Shows a quick overview of the user's thinking skill development.

Training Module

Provides structured exercises categorized by thinking type. Users can generate new AI-created exercises tailored to their specific thinking skills and current proficiency level.

Problem Solver

Allows users to input real-world problems they're facing and receive a step-by-step breakdown of the thinking process needed to solve them. This helps users see how effective thinking is applied to practical situations.

Analytics

Detailed charts and metrics showing progress, time spent, accuracy scores, and skill development over time. Helps users understand their strengths and areas for improvement.

Achievements

A collection of unlockable badges and rewards that users can earn as they develop their thinking skills and complete various challenges on the platform.

Thinking Process Tools

Advanced tools for evaluating thinking quality, reverse-engineering solutions, and verifying logical connections between problems and solutions. Helps users develop metacognitive abilities.

OpenAI Integration

ThinkTrainer leverages OpenAI's GPT models through several specialized functions:

  • generateThinkingProcess: Creates step-by-step thought processes for solving problems
  • evaluateThinkingProcess: Analyzes and scores user thinking with detailed feedback
  • generateExercise: Creates custom exercises tailored to specific thinking types
  • reverseEngineerThinking: Reconstructs the thinking needed to arrive at a given solution
  • verifyThinkingProcess: Validates whether a thinking process logically connects to its conclusion
Note: The application requires a valid OpenAI API key to enable these AI-powered features.

Data Model

ThinkTrainer uses a structured data model to track user progress and manage the application:

  • Users: User profiles with authentication information and progress level
  • ThinkingSkills: The four core thinking types with descriptions and visual attributes
  • UserSkills: Tracks each user's progress in each thinking skill
  • Exercises: Training activities categorized by skill type and difficulty
  • UserActivities: Record of all user interactions with the platform
  • UserProblems: Problems submitted by users with their generated thinking processes
  • Achievements: Unlockable rewards for skill development and platform usage
  • WeeklyActivity: Time-based tracking of platform engagement
